■6軸コプター概要
6軸に移動可能なマルチコプターを6個のプロペラで実現することを目標としている．
最終的には自動で衝突回避・PCからコマンドを受信しアクロバットな動きをさせる．

■開発フェーズ
ラフ設計段階
・電子回路
	0x基板(デバイスドライバ作成・動作確認用基板)動作確認済み
	1x基板作成中
・アクチュエータ
	構想段階(モータ・プロペラ仮選定済み)
・ソフトウェア
	デバイスドライバ実装完了．1x基板作成後制御実装．
・筐体
	構想段階(ラフ設計,実機未確認)

■構成
・プロペラ配置は3d図面及び./devlog/pictureを参照(※Z軸フレーム向き間違い．修正予定)
・軽量で高効率なコアレスモータとリポバッテリ(2セル直列)を使用
・移動量は3軸ジャイロ及び3軸加速度センサでモニタ
・EEPROMにチューニングパラメータを保存
・測距センサを全軸に付け，衝突を回避するようなロジックを入れる
・デバイスの電源電圧の都合上4.5V/3.6Vの2電源を生成．
  本来I/Fにレベルシフタを入れるべきだが抵抗で保護し実力で故障回避を狙う．
  そのため電源電圧を極力近づける設計とした．
